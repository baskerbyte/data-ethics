Main

Arquivo principal, responsável por mostrar o menu e handlar as opções

A partir do input válido recebido pela função menu, realizar a ação que o usuário desejar

Menu

A função menu será executada enquanto o usuário não informar um número válido, ou seja letras, menores que 0 ou maior que a quantidade de opções

Criar Perfil

Os inputs são divididos em etapas para não ficarem tão consativos para o usuário

O "do" é usado para evitar que o usuário digite algo errado e tenha que recomeçar todo o registro do começo

Campos do perfil

Para não poluir o código foi usado o tipo "Runnable" para setar alguma propriedade ao perfil do usuário

Se a operação não for bem sucedida ele emite um erro, esse erro é capturado no catch retornando true para continuar o loop até retornar uma interação válida

Editar perfil

Ele gera os detalhes do perfil para o usuário editar por linha (que serão enumeradas na função menu), depois ele pega a linha e separa ela no "-" para pegar somente o nome do field e não seu valor

Esses passos foram feitos para evitar ficar repetindo o código, seguindo a mesma lógica do criar perfil

Listar perfis

Chama a função de detalhes para cada usuário registrado

Detalhes do perfil

Gera os detalhes do perfil, se a opção expanded for verdadeira ele coloca ainda mais informações

Os fields precisam ser adicionados em uma ordem específica, para não quebrar a função de editar perfil

Deletar perfil

Pede algumas informações como email, senha e CPF para confirmar a exclusão do perfil
